# 注释插件

- github: https://github.com/numToStr/Comment.nvim

## 快捷键

### Normal Mode
- `gcc`: 注释或取消注释当前行
- `gbc`: 块注释
- `[count]gcc`: 指定行数注释
- `[count]gbc`: 指定行数块注释
- `gc[count]{motion}`: region注释
- `gb[count]{motion}`: region块注释

- `gcw`: 注释当前单词
- `gc$`: 注释光标位置到行尾
- `gc}`: 注释当前行到下一个空行
- `gca}`: 注释一对括号
- `gcip`: 段落注释

### Visual Mode
- `gc`: region注释
- `gb`: region块注释
